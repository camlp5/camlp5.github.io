Camlp5

Overview

Camlp5 is a preprocessor-pretty-printer of OCaml.
It is compatible with OCaml versions from 1.07 to 4.02.4.

The current distributed version is 6.16 (released 2016-04-29).

Distribution

You need OCaml to compile.

Download the sources: camlp5-6.16.tgz.

Browse the documentation (no major change in documentation from 6.00 to 6.16).

Or download the documentation:

  • In dvi format: camlp5-6.00.dvi
  • In ps format: camlp5-6.00.ps
  • In pdf format: camlp5-6.00.pdf
  • In texinfo format: camlp5-6.00.info.tgz
  • In html format: camlp5-6.00.html.tgz

Work in progress...

  • The development version of Camlp5 is browsable and downloadable from a git repository using command: 
          git clone https://gforge.inria.fr/anonscm/git/camlp5/camlp5.git
    This creates a directory named "camlp5" holding the sources. To update it, in case of changes, "cd camlp5" and type: 
          git pull
  • The future documentation is available as long as Camlp5 is improved.
    Warning: applies to the next release.
    Therefore, may contain future features not yet present in distributions.
