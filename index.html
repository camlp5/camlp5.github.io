Camlp5

Overview

Camlp5 is a preprocessor-pretty-printer of OCaml.
It is compatible with all versions of ocaml >= 4.00.0, and (theoretically) compatible with all versions of ocaml from 1.07 to 4.11.1 (when they compile), and jocaml 3.12.0 to 3.12.1, though the latter is not tested.

There are two distributed versions of Camlp5:
The current distributed version is 8.00 (released 2021-01-02). This version has many, many changes from the 7.xx releases, and hence we will also maintain a 7.xx-compatible release stream.
The current 7.xx-compatible release is 7.14 (released 2021-12-26).

Distribution

You need OCaml to compile.

Download the sources: Release 8.00 tgz.
Release 7.14 tgz.

Alternatively, checkout the Git repository from: camlp5/camlp5 (master branch)
camlp5/camlp5 (release 7.xx branch)

The documentation has also changed: the 7.xx branch documentation remains available
but new documentation is being written using Sphinx/RST, and is available here as well as at camlp5.readthedocs.io (where PDF, HTML, and EPub-downloadable versions are also available).

Or download backlevel documentation (no change since 7.12):

  • In dvi format: camlp5-7.12.dvi
  • In ps format: camlp5-7.12.ps
  • In pdf format: camlp5-7.12.pdf
  • In texinfo format: camlp5-7.12.info.tgz
  • In html format: camlp5-7.12.html.tgz

Work in progress...

  • The development version of Camlp5 is browsable and downloadable from a git repository using command: 
          git clone https://github.com/camlp5/camlp5
    This creates a directory named "camlp5" holding the sources. To update it, in case of changes, "cd camlp5" and type: 
          git pull
  • The future documentation is available as long as Camlp5 is improved.
    Warning: applies to the next release.
    Therefore, may contain future features not yet present in distributions.
